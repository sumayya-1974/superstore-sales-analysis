# superstore-sales-analysis
End-to-end sales analysis using Python and SQL
Project Overview
This project analyzes a Superstore sales dataset to draw meaningful business insights related to sales performance, product categories, delivery efficiency and customer demand.
## Tools & Technologies
- Python (Pandas, Matplotlib)
- SQLite (SQL queries)
- Excel  (Business reporting via exported analysis results)
- VS Code
## Dataset
- Source: Superstore Sales Dataset
- Records: ~10,000 orders
- Features include order details, product categories, regions, and sales values.
## Key Business KPIs Analyzed
1. Total Sales by Category  
2. Sales by Sub-Category  
3. Monthly Sales Trend  
4. Delivery Days Analysis by Ship Mode  
5. Top 10 Products by Sales  
## Key Insights (Summary)
- Technology category contributes the highest total sales.
- Standard Class shipping shows longer average delivery time compared to faster ship modes.
- A small set of products contributes disproportionately to total revenue.
